# Multi-Agent Disaster Response System

A SPADE-based multi-agent system for intelligent disaster detection and emergency response coordination.

## Architecture
- 7 Agent Types working in coordination
- XMPP-based communication
- Real-time disaster simulation
- Automated resource allocation

## Agents
1. Sensor Agent - Detects disasters
2. Command Center - Coordinates response
3. Fire Agent - Fire suppression
4. Ambulance Agent - Medical response
5. Evacuation Agent - Civilian rescue
6. Hospital Agent - Medical treatment
7. Traffic Agent - Route management

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run system: `python main.py`
